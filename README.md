# 📊 Dashboard Executivo de Vendas (Power BI)

## 📌 Sobre o Projeto

Este projeto consiste na criação de um **Dashboard Executivo de Vendas** desenvolvido no **Power BI**, com o objetivo de transformar dados brutos (fictícios) em **informações estratégicas para tomada de decisão**.

O dashboard apresenta indicadores importantes de desempenho, permitindo uma análise clara e rápida das vendas, tendências e resultados do negócio.

Este projeto foi desenvolvido durante um trabalho em grupo no segundo semestre da faculdade, demonstrando habilidades em **análise de dados, visualização e construção de dashboards interativos, além de trabalho em equipe e cooperação mútua**.

Os dados analisados são fictícios, foram gerados por Inteligência Artificial e pertencem a uma empresa de **vendas de produtos e serviços de tecnologia**, também fictícia.

---

## 🎯 Objetivos do Dashboard

* Monitorar o desempenho das vendas
* Identificar tendências e padrões nos dados
* Facilitar a análise de indicadores de negócio
* Apoiar a tomada de decisões baseada em dados

---

## 🛠️ Ferramentas Utilizadas

* 📊 Power BI
* 📁 Excel / Dataset de Vendas (fictício, gerado por IA)
* 📈 Técnicas de Data Visualization
